Zotero connection details

Group libraries

Use a Zotero group library where you want several people to contribute to a shared bibliographic library which can be synched with Heurist (or used independently when writing).

In order to synch data from a Zotero library, you need to know the User ID (for a personal library) or the Group ID (for a group library), and the Private key.

How to obtain IDs and keys

First log in to Zotero on the web. To obtain a Group ID, go to https://www.zotero.org/groups/ and click on the group. The Group ID is the numeric value (generally around 6 digits) in the URL displayed in the browser address bar.

To obtain a User ID and the Private Keys that you have created (if any), go to https://www.zotero.org/settings/keys. Your User ID is displayed near the top of the page followed by your Private Keys.

To get the value for an existing Private Key, click on the Edit Key link and copy the private key from the end of the URL shown in your browser address bar (it is an unreadable string of characters along the lines of: z6fCDq1BGyzvPd9MuhyTGbWK )

You can also create a new private key with the link Create new private key which will display the Private Key created and ask you to copy it.

How to use the ID and key

Go to the menu Database > Properties, and edit the Zotero field under Synchronisation and Indexing. Enter as Title,UserID,GroupID,Key eg. My library,1958388,,YjUxZzcgq1fhCd9YyxgzzVNX

Enter the UserID for a personal library or the Group ID for a group library. DO NOT enter both IDs - use a blank ( ,, ) for the ID which is not required.

Multiple libraries can be synchronised by separating the specifications with the pipe symbol ( | ) something like:
My library,1958388,,YjUxZzcgq1fhCd9YyxgzzVNX | XYZ project,,274864,DuFHBTyVYVhuIhttUotyqBSjb

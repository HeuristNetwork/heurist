Mapping Drawing

Header

Record can have geo field. Geo field contains coordinates that represent geographic feature: Point, Polyline, Polygon or their combination named Collection. These data are stored in database as WKT and represented on client side as GeoJSON.

Addition. Select drawing tool in top left corner of map. Cursor should be changed to 'Cross'. Add the desired feature by clicking on map. Double click to finish the drawing.

Edition Make sure that current tool is "hand" - selection tool. Select feature to be edited and add/drag vertices. To delete vertex right click and confirm deletion. To delete the selected feature from map click button on right panel.

Direct edition of coordinates. You may define coordinates explicitely. Coordinates of currently selected feature are displayed in the list on right panel. Modify them and then click 'Apply' button. To add new feature, select the desired type on drawing tools, enter coordinates and click 'Apply'

Note You may add/draw Circle and Rectangle features, however, WKT and GeoJSON do not support them. These types of features will be stored in database as Polygones.

Load external data. You may import geographic data from external sources. Supported formats are WKT, GeoJSON, KML, GPX, GeoRSS, Plain list of coordinates. Click 'Add geometry' to paste content or upload file of supported format

Note. All geographic features from external sources will be stored as geometry collection in the only record. Any non geographic information (properties) will be lost. If you wish to preserve it, import these data via Main Menu->Import. In latter case each feature will be imported as a separate record.

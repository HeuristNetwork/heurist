Cite-in-Word tool

What does it do?

Installing the bibliography citation tool places three buttons in a toolbar in Microsoft Word. Their functions are as follows:

  1. a reference search which finds references in the Heurist database and inserts them in your text;
  2. a bibliography builder which generates a formatted bibliography from those references at the end of your document;
  3. a configuration function.

Getting started


 

Use the pre-configured set of record types, customise them, borrow or add your own.


 
  • Adding data (records)
     
         Click the Add Record button above to add one of the default record types. more...
     
  • Customising
     
         Select Database | Manage Structure from the menus above.
         This will allow you to add, delete and modify the record types (entities) in the database,
         the fields (attributes) you record, and the terms used to populate dropdowns. more...
     
  • Borrowing
     
         Select Database | Acquire from databases
         Allows import of record types, fields and terms from any registered database. more...
     
  • Templates
     
         Select Database | Acquire from templates
         Detailed annotated descriptions will be displayed for a number of common database types. more...

Tip 3: Getting started

There are no records in this database.

You may wish to experiment with the pre-configured set of record types, customise them or start to add your own.

  • Adding records
    Click the Add New Record button above to add one of the default record types. See Help

  • Modify the current database structure
    Select the Database menu (above right) | Manage Structure. This will allow you to add, delete and modify the record types (entities) in the database, the fields (attributes)you record, and lists of terms. See Help

  • Import structure from templates
    Select Database | Acquire from templates. Detailed annotated descriptions will be displayed. See Help

  • Import structure from databases
    Select Database | Acquire from databases. Allows import of record types, fields and terms from any registered database. See Help

Using the data


The basic workflow for using Heurist data is to filter the database, which creates a subset of records, then generate output from those records. Output can be to the screen: a list of the filtered subset of database records in the centre panel, and various visualisations, reports and analysis in the righthand panel; or it can be printed or embedded (dynamically updated) in a web site.


Heurist provides a sophisticated set of filter capabilities - simple, text-based, categories, fuzzy matching, facet searches, spatial and temporal searches - along with the ability to create rules which extend the search results outwards through a network of linked records. Filter criteria and rules can be saved for re-use in a navigation tree in the lefthand panel.

The HuNI Project

The HuNI project (Humanities Networked Infrastructure, http://huni.net.au) has built a central searchable aggregate of metadata harvested from 28 Australian cultural datasets, including AusStage, AusLit, Dictionary of Art and Architecture Online, Australian Dictionary of Biography, Circus Oz, Paradisec and Dictionary of Sydney.

Heurist is providing a database-on-demand function for HuNI. Databases created using the HuNI Core Metadata template can be immediately exported to a HuNI harvestable format. Other databases (including the Dictionary of Sydney, which was set up in Heurist before the existence of the HuNI Core) can be made available for harvesting by mapping fields to the HuNI Core naming conventions.

HuNI was funded as a Virtual Laboratory by the Australian Federal Government National eResearch Collaboration Tools and Resources Project (NeCTAR in 2013 - 2014) and is maintained by Deakin University.

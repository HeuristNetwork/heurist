Namespace TimeMap.intervals

Defined in:
timemap.js

Lookup map of common timeline intervals. Add custom intervals here if you want to refer to them by key rather than as a function name. 

    TimeMap.init({
        bandIntervals: "hr",
        // etc...
    });

Heurist Academic Knowledge Management System (C) 2005 - 2015 University of Sydney

An Open Source collaborative web database for recording and publishing richly interlinked research
data integrated with web bookmarking, bibliographic functions, mapping, timelines and network graphs

A simple one minute registration allows you to use Heurist on this server (after approval by the system adminstrator).
Create your first database (you must first register as a user). Subsequent databases can be created with New Database when logged into an existing database. You become the owner of the new databases created.
Browse the list of databases on this server - you will only be able to open databases for which you have login credentials.
A one-stop shop for everything Heurist, including full documentation, training materials, workshops, hosted Heurist services, exemplars and annotated database templates.

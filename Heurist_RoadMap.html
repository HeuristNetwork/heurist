Heurist RoadMap

This page describes planned new features and development priorities for the Heurist software ( HeuristNetwork.org ).

This diagram represents the main priorities for 2024 (some have already been done as of Nov 2023, as indicated). For each section it implies likely development order from left to right, but the order may change as users/projects request or dictate. This does not cover routine maintenance, small interface improvements and bug fixing.

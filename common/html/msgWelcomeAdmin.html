Database Administration

Where the user has the right administrative privileges - generally an Admin on the Database Managers group - this page will allow them to:

  • Modify the structure of the database
  • Clone, delete and clear records from the database.
  • Add new users and workgroups and control access.
  • Register the database with the central index at HeuristScholar.org (database owner only).
  • Carry out various database consistency verification and maintenance tasks.
  • Run special functions related to specific projects such as HuNI and FAIMS.

Note: the database creator is automatically the owner of the database and an Admin in the Database Managers group

Please select functions in the menu on the left.

Note that the most commonly used database administration functions are available under Database in the menus at top right of the Main Page.

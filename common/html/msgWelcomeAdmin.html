Database Administration Switchboard

This screen allows Heurist users to create new databases on the server running this software (unless disabled by system adminstrator) and to register them with the central index at HeuristScholar.org

Where the user has administrative privileges (the database creator is automatically the owner) it will allow them to:

  • edit the database structure and import structural elements from other Heurist databases;
  • add new users and user groups and control access ;
  • carry out various database consistency verification and maintenance tasks.

Please select functions in the menu on the left.

Services configured here enable the lookup of external resources such as a gazeteer, a thesaurus or a library catalogue from within a data entry from and insert of
one or more fields derived from the external resource into the data. They can also be used to provide specialised processing such as predictive setting of keywords
based on frequency of usage and matching with external resources.

Existing services can be configured here, while new services can be programmed using the template and guidelines provided in the source code
(please contribute them to the gitHub repository if likely to be of general use).

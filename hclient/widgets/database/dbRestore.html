This function restores a database from a safeguard file in Zip format created either by deletion of a database, by nightly backup/archiving functions, or by requesting download of an safeguard file through Publish > Safeguard file.

If restoring from a file created on another server, we recommend uploading it to /srv/transfer using sftp. Older archives (pre 2024) in tar.bz2 will need to be unzipped and loaded manually via the Unix command line.

After successful completion you will be required to login into the restored database.


Directory:    /hclient/widgets/database

Overview:    Heurist database management widgets

Notes:

dbManager.js - list of all databases where current user is an owner, or server manager
dbAction.js  - popup dialog or widget to define action parameters, send request to server, show progress and final report

content of dialog for various actions

dbClear.html
dbClone.html
dbCreate.html   
dbDelete.html
dbRename.html
dbRestore.html

Localized version are with lang suffix (dbCreate_fre.html)


dbValidate.html - verify integrity

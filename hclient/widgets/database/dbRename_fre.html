Cela exécutera les fonctions de clonage et de suppression dos à dos afin de renommer la base de données actuelle. Veuillez donc vous assurer que toutes les modifications/changements ont été enregistrés avant de continuer.

Une fois l'opération terminée avec succès, vous devrez vous connecter à la base de données nouvellement clonée.

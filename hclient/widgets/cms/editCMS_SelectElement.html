A group is normally used to keep several other elements togather so that they can be displayed and scaled across the page, for example a search on the left, results in the middle and a custom report or map on the right.

Without groups, all the elements will normally be displayed one above the other.
The simple text box contains WYSIWYG text which can be edited with the embedded TinyMCE text editor. This allows for headings, font styles, quotes, hyperlinks, etc...

Embedded media referenced by the database (whether local to the database or a remote reference) will be rendered in place in the page.

The simple text box may occupy the whole width of the page or be placed within a container or other element to be scaled in width and/or sit next to other elements.

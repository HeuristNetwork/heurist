This function allows one to design a layout displaying a connected set of Heurist widgets (search, saved filters, results lists, custom reports, maps, timelines, network diagrams etc.) which can be embedded in a third-party website.

Create a new layout or select an existing one from the dropdown (existing layouts can be edited).

Owing to the rich functionality and interconnection of Heurist widgets and cross-domain access restrictions, embedding widgets directly in the html of an external site is complex and could lead to long-term instability due to code changes. By populating an iframe with a layout maintained in Heurist, we can ensure that the functionality in this layout is maintained as part of the Heurist codebase.

The appearance and behaviour of the embedded functions can be modified by editing the layout within the Heurist database (this will alter all web pages which use the layout - create several layouts if required).

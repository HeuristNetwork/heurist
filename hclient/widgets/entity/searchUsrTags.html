Use controls above to filter and sort visible set of tags.

Available tags on left side. Selected tags are on right side. To select a tag just click on it. Or you may type name of tag in input field on right side and press Enter. If tag with such name exists it will be selected, otherwise new tag will be created and selected. Note: selected set is preliminary. To assign tags to record click Save on record edit form.

To manage your tags (description, deletion, usage) select in your user menu "My tags".

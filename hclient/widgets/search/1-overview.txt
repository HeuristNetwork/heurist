
Directory:	/hclient/widgets/search

Overview:   Files in this directory manage searching/filtering of the database, as well as the management of the navigation panel containing saved searches,
            and fuctions to build search (filter) strings and Rules

            n_svstree.json provides the structure for folders and faceted searches within the navigation panel for user 'n'
            This structure is stored in the users/groups table
            The navigation panel is built up using all the svstree structures which apply to the current user

Updated: 	18 Oct 2015

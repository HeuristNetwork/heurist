Heurist code update

The Heurist code has been updated from version 4.x.x to version 4.x.y since you last accessed it. This update provides routine bug fixing and minor enhancements - your database will not be affected.

Please clear the browser cache and reload this page to refresh the code before proceeding. If you do not clear the cache the interface will misbehave.

  • Chrome: Ctrl-Shift-Del, check Cached images and files, click Clear Browsing Data button
  • Edge: Ctrl-Shift-Del, check Cached data and files, click Clear button
  • Firefox: Ctrl-Shift-Del, open up Details, check Cache, click the Clear Now button
  • Safari: Ctrl-Alt-E, click the Empty button
(there is no need to clear cookies, only downloaded data and files)

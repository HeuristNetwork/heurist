Heurist code update

The Heurist code has been updated from version 4.x.x to version 4.x.y since

you last accessed it. Updates provide routine bug fixing and enhancements.

Your database will not be affected.
 

Please reload with Ctrl F5, or clear the browser cache and reload this page

 
If you do not clear the cache old code hangs around and the interface will misbehave.

  • Chrome: Ctrl-Shift-Del, check Cached images and files, click Clear Browsing Data button
  • Edge: Ctrl-Shift-Del, check Cached data and files, click Clear button
  • Firefox: Ctrl-Shift-Del, open up Details, check Cache, click the Clear Now button
  • Safari: Ctrl-Alt-E, click the Empty button
Note: do not clear cookies, only downloaded data and files. On Macs: Ctrl = option, alt = command.

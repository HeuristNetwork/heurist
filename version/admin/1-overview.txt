Directory:	/admin

Overview:

This directory contains all the adminstrative interface code from Heurist Vsn 3, which is used by Heurist Vsn 4.
The code has been modified where needed to work within H4
The most important menus have been moved into the main search/filter page layout
This page is slated for elimination once all functions moved as submenus of the main page menus

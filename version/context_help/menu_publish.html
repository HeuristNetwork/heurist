The Publish menu delivers easily edited CMS websites embedded directly within the database, as well as individual web pages which can be embedded in other sites. It also provides an archiving fucntion which enables the download of a complete, documented copy of the data in the database in open format, and export functions (also available directly in Explore).

Publish menu

Functions for managing CMS websites, webpages, export and archiving.

  • Website
    • Create - sets up a new CMS website. A database can have multiple websites for different audiences
    • Edit - edit an existing CMS website stored in the database
    • View - view an existing CMS website in a separate window - use to check results and to obtain the URL
    • Standalone web page - create or edit a CMS-generated web page for embedding in a third-party website
  • Export
    • CSV – comma or tab-separated file with a selection of fields, potentially from multiple connected record types
    • XML – comprehensively documented XML which can be imported automatically into any Heurist database
    • JSON – a native javascript format which also be used for transferring data between databases
    • GeoJSON – a javascript format for exporting spatial objects to GIS
    • KML - an XML format for exporting spatial objects and attributes to GIS
    • GEPHI - exports GEFX files of node and edge data for use in GEPHI network analysis software
  • Safeguard
    • Archive package - a fully internally documented archive package of all the data in the database

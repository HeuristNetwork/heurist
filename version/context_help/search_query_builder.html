Query Builder

Predicates

Predicate is base element of query. It consists of search token (what to search) and search value.
Examples JSON: {"t":25} {"f:103":"Find me!"} {"likedto:90":[{"ids":987,443,543}]} {"related_to:2098":"987,443,543"} Plain text t:25

Search tokens

Possible tokens are:

  • ids, id: record ID
  • title: reord title
  • url, u: record url
  • notes, n: record notes (`rec_ScratchPad`)
  • added: creation date
  • date, modified: edition date
  • after, since, before: aliases for added:>date and added<date
  • addedby: added by specified user (`rec_AddedByUGrpID`)
  • owner,workgroup,wg: owner of record (`rec_OwnerUGrpID`)
  • tag, keyword, kwd: tag name (`usrtags`.`tag_Text`)
  • user, usr: bookmarked by user (`usrbookmarks`.`bkm_UGrpID`)
  • t,type: record type
  • f,field: field type id
  • linked_to,linkedfrom,related_to,relatedfrom,links: various link predicates
Note: for "field" and "links" token is combination of token name and field or relation type identificator. For example:
  • linked_to:90 - linked via field #90
  • relatedfrom:2340 - related via relation type (term) #2340

Simple search

The search field above the menu searches for People using all fields in the database including the person's names, places, schools, universities and colleges, and textual descriptions. The search is a simple text search, is not case sensitive and partial matches will be found. All entered words must be matched, although the order does not matter.

For example, to find Eleanor Alice Allen you could enter the full name or Eleanor Allen or Allen Eleanor. Shorter strings will tend to be less discriminating - Eleanor will find this person, but Ele finds 47 people.

To search for an exact phrase, use double quotation marks "". Please note that in this case the phrase must be exactly as it appears in the database including the same punctuation. For example: "St. Paul's College" (note the full-stop after St. and the apostrophes).

Facets

The results can be refined using the 'facets' (database values) on the left hand side of the search results page. The facets allow you to refine your search according to eg university, qualification, school. gender or place of birth. Different types of facet as shown below - list, slider, search or dropdown - can be used depending on the type and number of values involved.

When you apply an additional filter to your original search using one of the facets, all the other facets will adjust to show the number of people who still satisfy each of the criteria (number in a circle).

Use the return arrow icon next to a selected facet to undo that filter, or the Reset filters button to undo the filtering by all the facets - this will leave your original text search in place.

Use filter criteria to narrow the list of tags.
Click on tag to edit it. If entered tag already exists it will be merged with other one.
Available tags on left side. Selected tags are on right side. To select a tag just click on it.
Or you may type name of tag in input field on right side and press Enter.
If tag with such name exists it will be selected, otherwise new tag will be created and selected.
Note: selected set is preliminary. To assign tags to record click Save on record edit form.

Heurist Academic Collaborative Database Version 3

Open Source collaborative web database software for recording and publishing
research data integrated with web bookmarking, bibliographic functions, mapping and timelines.



     Opens the shared database at Heurist Scholar.org

Register as new user
A simple two minute process will get you using our shared database in which you can create
and share (or not) internet bookmarks, bibliographic data and any other form of research data you wish.

Introductory tour
A quick overview of Heurist's field of application and capabilities

Full documentation
Comprehensive end-user documentation is contained in the online help system. The system is structured into
sections at different technical levels. It covers every aspect of Heurist from undeerstanding its domain
of application and the way it structures data to advanced topics in writing new output formats.

Example applications
A Heurist search returning links to the websites of a selection of Heurist-based projects

Publications
A bibliography of published papers, seminar and conference presentations (in preparation).

Namespace TimeMap.themes

Defined in:
timemap.js

Pre-set event/placemark themes in a variety of colors. Add custom themes here if you want to refer to them by key rather than as a function name. 

    TimeMap.init({
        options: {
            theme: "orange"
        },
        datasets: [
            {
                options: {
                    theme: "yellow"
                },
                // etc...
            }
        ],
        // etc...
    });

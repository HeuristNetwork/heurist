Namespace TimeMap.loaders

Defined in:
timemap.js

Namespace for different data loader functions. New loaders can add their factories or constructors to this object; loader functions are passed an object with parameters in TimeMap.init(). 

    TimeMap.init({
        datasets: [
            {
                // name of class in TimeMap.loaders
                type: "json_string",
                options: {
                    url: "mydata.json"
                },
                // etc...
            }
        ],
        // etc...
    });

Static Method Details
TimeMap.loaders.cancel(callbackName)
Cancel a specific load request. In practice, this is really only applicable to remote asynchronous loads. Note that this doesn't cancel the download of data, just the callback that loads it.
Parameters (optional)
callbackName {String} Name of the callback function to cancel
TimeMap.loaders.cancelAll()
Cancel all current load requests.

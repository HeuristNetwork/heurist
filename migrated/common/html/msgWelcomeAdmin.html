Database Administration Switchboard

This screen allows Heurist administrative users to carry out various actions on databases on the server running this software.

Where the user has the right administrative privileges (the database creator is automatically the owner) it will allow them to:

  • Register the database with the central index at HeuristScholar.org (database owner only)
  • Clone, delete and clear records from the database
  • Add new users and user groups and control access
  • Carry out various database consistency verification and maintenance tasks
  • Run special fucntions related to specific projects such ac HuNI and FAIMS

Please select functions in the menu on the left.

Other database management functions are available in the menus at top right of the home/search page.
